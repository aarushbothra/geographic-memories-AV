# Instructions for Opening in Max/MSP
## Download Prerequisites
1. Download [Spat 5][https://forum.ircam.fr/projects/detail/spat/]. They will ask you to make an account.
2. Extract the file. Copy the spat-x64 folder and paste it into Documents>Max 9>Packages.
3. Open Max. Go to File>Show Package Manager and search for CNMAT Externals. Download and install that.

## Opening and running
1. Clone this repo
2. In Max, go to File>Open and open the geographic_memories.maxproj file in the repo
